## Current scope

The library currently focuses on:

- `MapLibre` root map component
- Layer/source composition (`MapLibreSource`, `MapLibreLayer`, `MapLibreSourceGeojson`)
- Marker/popup composition (`MapLibreMarker`, `MapLibrePopup`)
- Control wrappers (`MapLibreControl`, `MapLibreNavigationControl`)
